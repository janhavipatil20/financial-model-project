# Financial Modeling Project

## Overview
This project is a financial model built using publicly available financial statements of a company. The model analyzes financial performance and estimates valuation using different financial techniques.
The model integrates all three financial statements to provide a complete financial analysis.

## Key Components
- Income Statement, Balance Sheet, Cash Flow Analysis
- Free Cash Flow calculation (FCFF & FCFE)
- Weighted Average Cost of Capital (WACC)
- Economic Value Added (EVA) & Market Value Added (MVA)
- Ratio Analysis and Interpretation

## Objective
To understand company performance and estimate its value using financial modeling concepts.

## Data Source
Publicly available financial statements (Annual Reports)

## Tools Used
- Microsoft Excel

## Key Learnings
- Linking financial statements
- Applying valuation techniques
- Interpreting financial ratios and performance
